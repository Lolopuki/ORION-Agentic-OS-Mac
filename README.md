# O.R.I.O.N. (Operativo de Respuesta e Inteligencia Organizada en Network)

O.R.I.O.N. is an open-source initiative focused on developing a local, agentic AI operating system designed for advanced task automation and recursive self-improvement on macOS architectures. Moving beyond the limitations of simple conversational assistants, O.R.I.O.N. aims to bridge the gap between high-level reasoning and low-level system execution.

  <b>Inspired by Jarvis. Built for the future of Autonomous Intelligence.</b>
</p>

## Project Overview

Our core philosophy is **recursive improvement**. O.R.I.O.N. utilizes a unique "Multi-Brain architecture" to safely evolve its own code while maintaining system stability through automated sandboxing and validated backups.

The project is currently in early alpha, running locally on specialized Apple Silicon hardware (MacBook Pro/M3 Max).

## Key Features

* **Acoustic Wake-Word "ORION":** Custom-tuned local detection for precise activation on macOS.
* **Task Execution Core:** Native integration with macOS system commands for controlling applications, managing files, and executing local scripts.
* **Multi-Model Orchestrator:** Dynamic routing between real-time executors (like Gemini 2.0 Flash) and deep reasoning architects (like Claude Sonnet/Opus 4.6).
* **Self-Correction Loop (Recursive Improvement):** A specialized pipeline that allows O.R.I.O.N. to:
    1.  Analyze its own execution logs or core code.
    2.  Propose architectural improvements or bug fixes in a sandbox.
    3.  Validate changes (syntax checks, core unit tests).
    4.  Execute automated `rollback` (backup) protocols before safely deploying the new code.

## Current State & Development Focus

This repository contains the stubs and foundational orchestrator for the project. Our primary development focus is creating a robust, failsafe pipeline for model-driven recursive self-refactoring.

We are currently optimizing the interface with advanced high-reasoning models to act as the core **"Architect"** brain for safe code generation and self-diagnostics.

## Tech Stack (Planned & Current)

* **Node.js / JavaScript:** Central Orchestrator & System Logic.
* **Python:** Acoustic Processing, Speech-to-Text, Local Model Interface.
* **Vertex AI (Google Cloud):** Integrated model endpoints (Planned connection to Claude 3.5 series via Vertex).
* **macOS APIs:** Local command execution.
